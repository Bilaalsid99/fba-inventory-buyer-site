# FBA Inventory Buyer Landing Page

A clean Next.js App Router landing page for a UK independent inventory buyer reviewing selected Amazon FBA UK removal and unfulfillable inventory lists.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
